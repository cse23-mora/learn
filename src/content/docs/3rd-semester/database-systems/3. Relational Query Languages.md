Declarative and Procedural
## Relational Algebra Operators
- Select: $\sigma$
- Project: $\sqcap$
- Union: $\cup$
- Set difference: $-$
- Set intersection: $\cap$
- Cartesian product: $\mathsf{X}$
- Rename: $\rho$

### Selection of tuples (rows)
for relation **r**:

$$\LARGE{\sigma_{A=B\ and\ D > 5}\ (r)}$$
